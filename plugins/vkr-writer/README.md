# vkr-writer

Two skills for writing Russian university theses (ВКР) that pass anti-plagiarism checks with 0% AI detection.

## Skills

- **academic-writer** — bachelor's thesis (бакалаврская ВКР): all sections, UML diagrams, economic chapter
- **master-thesis** — master's dissertation (магистерская диссертация): scientific register, literature review, novelty, formulas

## Install

```bash
claude plugin marketplace add YOUR_USERNAME/vkr-writer
claude plugin install vkr-writer@vkr-writer
```
